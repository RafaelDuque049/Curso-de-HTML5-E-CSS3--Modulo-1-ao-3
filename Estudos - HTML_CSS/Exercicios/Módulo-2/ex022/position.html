Os tipos de posições de elementos

static

O valor static, é o valor padrão que todos os elementos recebem assim que adicionados ao arquivo. Ela segue o conceito de os elementos se posicionar seguindo as posições que é adicionada ao HTML, ou seja, em forma de pilha.

relative

O valor relative, segue o mesmo conceito do static, mas trabalha em conjunto com outros 4 declarações, sendo eles: top(a partir do topo), bottom(a partir de baixo), left(a partir da esquerda) e right(a partir da direita). Quando essas declarações recebem algum valor, a parte fisíca do elemento continuara em sua mesma posição como se não tivesse se movido, mas a parte visual se moverá para a posição solicitada.

absolute

O valor absolute se posiciona a partir do seu elemento pai que esteja recebendo um valor position diferente de static. Uma curiosidade que, mesmo o elemento se posicionando no corpo, o resto da página age como se ele não existisse, sendo possível posiciona-lo onde deseja.

fixed

Ele se posiciona a partir do viewport da página, ou seja, da tela 100% do navegador, não se limitando a nenhuma tag do body.

sticky

O elemento sticky é como se fosse um hidrído entre o relative e fixed. O elemento sticky tem a habilidade de 'grudar' no viewport, isso significa que, quando você passa por esse elemento, assim que ele chegar ao topo da tela, ele se mantem ali, e quando você voltar para cima, ele retorna para a sua posição definida.
